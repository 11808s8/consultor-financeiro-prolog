# Consultor Financeiro em Prolog
Desenvolvido para a disciplina de Inteligência Artificial I da Universidade de Caxias do Sul (2/2019)

Para executar ele local:
Precisa ter o <a href="www.swi-prolog.org/Download.html">SWI-PL</a> instalado.

No terminal (testado em sistemas Unix apenas):

`swipl -s main.pl`

Após isto, insira o comando: 
`menu.`
